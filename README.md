A stunning, fully responsive portfolio website built with React, Tailwind CSS, and Framer Motion. Features smooth animations and design patterns used across the project.

## Features

- Futuristic Cyber Theme with neon gradients and glowing effects
- Smooth Framer Motion animations with scroll-triggered reveals
- 3D hover effects and interactive elements
- Floating particles and animated background elements
- Glass morphism design patterns

## How to run locally

1. git clone https://github.com/sidhiqsidhu/portfolio_new.git
2. cd portfolio_new
3. npm ci
4. npm start

## Build & deploy

- Build: `npm run build`
- Deploy to gh-pages: `npm run deploy` (the repo includes `gh-pages` in devDependencies)
